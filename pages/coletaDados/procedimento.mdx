# Etapas da coleta de dados

Todo o processo de coleta de dados foi realizado no Hospital Albert Einstein, em Goiânia. Uma enfermeira qualificada foi responsável por realizar as seguintes etapas:

<br/>

- [x] colocar o carrinho móvel de coleta de dados em qualquer parte do centro cirurgico do hospital
- [x] selecionar o voluntário entre colegas enfermeiras ou médicos
- [x] solicitar ao voluntário a assinatura do Termo de Consentimento Livre e Esclarecido
- [x] registrar as características do voluntário no Formulário do Gesto de Lavagem das Mãos
- [x] demonstrar ao voluntário o gesto que ele deve realizar 
- [x] gravar o gesto executando o script do software na pasta correspondente
- [x] registrar o gesto como concluído e siguir com o novo gesto
