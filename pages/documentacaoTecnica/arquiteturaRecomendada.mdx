# Arquitetura recomendada

1. **Processador:**
   - Intel Xeon Processor 2.4Ghz, compatível com arquiteturas 32-bit e 64-bit, e configuração de CPU Dual Core.
   - Alternativamente, é possível optar pela compatibilidade com o chip M1 de 2020, que apresenta uma CPU com 8 núcleos e uma GPU também com 8 núcleos.

2. **Placa Gráfica:**
   - Placa gráfica integrada com mínimo de 2GB de memória.

3. **Memória RAM:**
   - Mínimo de 8GB de RAM.

4. **Armazenamento:**
   - Mínimo de 50GB destinados ao Sistema Operacional e instalação.
   - Mínimo de 10GB de espaço livre em disco para armazenamento de modelos e dados não voláteis.
   - Espaço adicional de, no mínimo, 10GB para armazenamento de aproximadamente 10.000 imagens.

5. **Sistema Operacional:**
   - Ubuntu 16.04.7 LTS, xenial.
   - Alternativamente, é possível optar pela compatibilidade com o macOs Ventura 13.4.1.

6. **Sistema de virtualização para pacotes de aplicação:**
   - Docker versão 18.09.7, build 2d0083d.
   - Docker Compose version v2.23.0-desktop.1.

<p style={{textAlign: 'justify', textIndent: '20px', marginBottom: '5px'}}> Ao aderir às especificações mínimas recomendadas, você assegurará uma experiência estável e eficiente ao utilizar a aplicação. Essas configurações garantem um desempenho adequado, possibilitando o manuseio eficiente de modelos de inteligência artificial e a manipulação de imagens. </p>
